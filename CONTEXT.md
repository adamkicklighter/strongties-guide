## Project Overview

You are assisting with **StrongTies Guide**, an open, public, AI-assisted user guide designed to help people network more effectively and authentically during career development and job searches.

The guide operationalizes the networking framework presented in ["Deep Professional Networks: Building Meaningful Connections for Career Development"] by Adam Kicklighter.

**The core idea:** Networking can be treated as a structured, iterative process, similar to optimizing paths in a professional graph, rather than as ad-hoc social outreach or transactional cold asks.

**Project repository:** https://github.com/adamkicklighter/strongties-guide

---

## Target Audience

### Primary Users
- Job seekers and career switchers
- Data professionals, engineers, analysts, and knowledge workers
- People who dislike "traditional networking" but value thoughtful, intentional relationship-building

### Secondary Users
- Students and early-career professionals
- Anyone seeking a repeatable, low-anxiety framework for professional networking

**Assumption:** Users are assumed to be intelligent but not necessarily technical. Graph and ML metaphors should be used for intuition, not mathematical rigor.

---

## Core Mental Model (Do Not Deviate)

The entire guide is built on this metaphor:

> **Your professional network is a graph. Networking is message passing. Job opportunities emerge through attention-weighted paths mediated by trust.**

### Key Mappings from the Presentation

- **Nodes:** You, Companies, Roles, Hiring Managers, Connections
- **Edges:** Employment relationships, mutual connections, stated intent
- **Edge weights:** Strength of relationship, organizational proximity, credibility, trust
- **Message passing:** Outreach, conversations, referrals, follow-ups
- **Training objective:** Increase the likelihood that a hiring manager meaningfully considers you for a role, via trusted intermediaries
- **Convergence:** Visibility + trust inside the hiring network

This framing comes directly from the slides on:
- Professional Network Graphs
- Message Passing
- Training Objective
- Role of Edge Weights
- Message Passing as Network Strategy

**Never reduce this to "apply to more jobs" or "optimize your LinkedIn profile" alone.**

## Philosophical Constraints (Very Important)

When helping create content, prompts, or flows:

### 1. Networking is not transactional
- Users are **never** told to ask for jobs
- Users ask for advice, insight, perspective, or introductions

### 2. Clarity precedes outreach
Prerequisites (not optional steps):
- Objective statement
- Target markets
- Target companies

### 3. The user is marketing themselves, not begging
- Resume = marketing document
- Objective statement = positioning
- Outreach = professional signal, not a favor request

### 4. Rejection is a pivot, not a failure
- "No" is treated as gradient feedback
- Users adjust strategy, not self-worth

### 5. Long-term orientation
- The guide emphasizes "always be networking," not just job-search bursts

> All of these principles are explicitly stated or implied throughout the latter half of the presentation (Marketing Plan, Right Mindset, Working the Process, Referrals, Always Be Networking).
